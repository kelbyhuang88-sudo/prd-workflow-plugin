# 前端组件配置模板

> 本模板用于记录模块的前端组件信息，供AI快速定位目标文件

**使用说明：**
1. 复制此模板到 `{需求类型}/组件配置/{模块名}-前端组件.md`
2. 根据实际前端代码填写信息
3. 定期维护更新

## 页面清单

| 页面名称 | 文件路径 | 功能说明 |
|----------|----------|----------|
| 列表页 | `src/views/{module}/List.vue` | 数据列表展示、查询、筛选 |
| 编辑页 | `src/views/{module}/Edit.vue` | 新增/编辑表单 |
| 详情页 | `src/views/{module}/Detail.vue` | 数据详情展示 |

## 表单字段

**页面：** {如：订单编辑页}

| 字段名 | 字段类型 | 是否必填 | 数据来源 | 备注 |
|--------|----------|----------|----------|------|
| 订单号 | input | 是 | 手工输入 | |
| 客户名称 | select | 是 | 下拉选择 | 接口获取选项 |
| 创建时间 | date-picker | 否 | 选择 | 默认当天 |
| 状态 | select | 否 | 下拉选择 | 固定选项 |

---

## 按钮操作

**页面：** {如：订单编辑页}

| 按钮名称 | 触发事件 | API调用 | 说明 |
|----------|----------|----------|------|
| 保存 | `onSave()` | `POST /api/order` | 新增/修改提交 |
| 提交审批 | `onSubmitApproval()` | `POST /api/order/approval` | 提交OA审批 |
| 驳回 | `onReject()` | `POST /api/order/reject` | 驳回订单 |
| 删除 | `onDelete()` | `DELETE /api/order/{id}` | 删除订单 |

---

**页面：** {如：订单列表页}

|------|----------|------------|------------|------|

| 状态 | status | 是 | 是 | 标签显示 |
| 创建时间 | createTime | 是 | 否 | 日期范围筛选 |
| 操作 | - | 否 | 否 | 操作按钮列 |

## API接口

| 接口名称 | 请求方式 | 路径 | 说明 |
|----------|----------|------|------|
| 列表查询 | GET | `/api/{module}/list` | 分页查询 |
| 详情查询 | GET | `/api/{module}/{id}` | 单条查询 |
| 新增 | POST | `/api/{module}` | 新增提交 |
| 修改 | PUT | `/api/{module}/{id}` | 修改提交 |
| 删除 | DELETE | `/api/{module}/{id}` | 删除 |
